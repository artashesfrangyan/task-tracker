# Простой таск-трекер. 
На моей странице github есть вариант трекера задач намного сложнее, с DND и бэкэндом на GO (для запуска требуется Docker): github.com/artashesfrangyan/avito

# Инструкция по установке и запуску: 
1) скопировать репозиторий
2) открыть папку проекта в терминале и написать: 
>> npm install
>> npm run dev
