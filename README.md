# Простой таск-трекер. 
На моей странице github есть вариант трекера задач намного сложнее, с DND и бэкэндом на GO (для запуска требуется Docker): github.com/artashesfrangyan/avito

Здесь данные генерируются прямо на frontend — каждую секунду задачи добавляются, обновляются или удаляются. Это имитация работы с реальным бэкендом.

🔗 **Живое демо**: [https://artashesfrangyan.github.io/task-tracker/](https://artashesfrangyan.github.io/task-tracker/)

## Стек

- React + TypeScript
- Redux Toolkit (управление состоянием)
- Material UI (компоненты)
- Vite (сборка)
- GitHub Pages + Actions (автодеплой)

## Как запустить локально

1. Клонировать репозиторий:
```bash
git clone https://github.com/artashesfrangyan/task-tracker.git
```

2. Перейти в папку проекта:
```bash
cd task-tracker
```
3. Установить зависимости:
```bash
npm install
```
4. Запустить дев-сервер:
```bash
npm run dev
```
4. Открыть http://localhost:5173 в браузере.

Как проверить

· На главной странице (/) отображается список задач.
    Каждую секунду происходят изменения: может появиться новая задача, у существующей поменяться статус (создано → в процессе → завершено) или какая-то задача исчезнуть.
· На странице статистики (/stats) видно общее количество задач и разбивку по статусам. Цифры обновляются автоматически.
